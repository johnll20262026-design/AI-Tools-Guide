# AI工具指南 - 需求拆解文档

## 产品概述

- **产品类型**: AI 教程指南网站（落地页/官网）
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: 中文AI工具学习者、开发者、技术从业者
- **核心价值**: 面向中文用户的AI工具实战教程与落地指南，提供系统化的AI工具学习路径
- **界面语言**: 中文
- **主题偏好**: 深色（深灰/黑色背景 + 绿色/青色强调色）
- **导航模式**: 锚点导航（单页滚动）

---

## 页面结构总览

**页面文件**: `HomePage.tsx`

| 区块名称 | 锚点 | 区块说明 |
|---------|------|---------|
| 顶部导航栏 | `#nav` | 固定顶部导航，含品牌Logo+站点名+导航链接 |
| 网站标题区 | `#hero` | 主标题"AI工具指南" + 副标题 + 简介描述 |
| 核心教程分类卡片区 | `#categories` | 6-8个教程分类卡片，卡片式网格布局 |
| 数据统计展示区 | `#stats` | 教程数量、案例数量、配置教程等核心数据展示 |
| 学习路径推荐区 | `#paths` | 3条不同起点的学习路径推荐 |
| 实战案例展示区 | `#cases` | 精选典型AI应用场景案例展示 |
| 页脚信息 | `#footer` | 版权信息、链接等 |

---

## 页面布局建议

- **布局模式**: 单栏垂直流式布局，各区块依次堆叠，区块间留白充足
- **视觉重心**: 标题区（Hero）为视觉焦点，卡片区为内容核心
- **结果承载区**: 本页面为纯展示型内容，无用户输入/生成结果，各区块内容均为静态展示

## 功能列表

- **区块**: 顶部导航栏 (`#nav`)
  - **页面目标**: 提供页面内快速跳转，固定顶部始终可见
  - **功能点**:
    - 品牌Logo + 站点名展示（左侧）
    - 导航链接锚点跳转（平滑滚动）
    - 移动端响应式汉堡菜单

- **区块**: 网站标题区 (`#hero`)
  - **页面目标**: 传达网站核心定位，建立第一印象
  - **功能点**:
    - 主标题"AI工具指南"大字号展示（可做渐变效果）
    - 副标题"面向中文用户的AI工具实战教程与落地指南"
    - 简短介绍文案（1-2句话说明网站价值）

- **区块**: 核心教程分类卡片区 (`#categories`)
  - **页面目标**: 展示教程分类，引导用户按需浏览
  - **功能点**:
    - 6-8个分类卡片，网格布局（桌面端3-4列，移动端1-2列）
    - 每个卡片含：分类标题 + 简短描述（1-2句话）
    - 卡片hover微交互（边框高亮/轻微上浮）
    - 分类建议：Prompt工程、AI绘画、AI视频、AI编程、AI办公、AI Agent、模型部署、AI安全

- **区块**: 数据统计展示区 (`#stats`)
  - **页面目标**: 用数据增强信任感，展示内容规模
  - **功能点**:
    - 3-4个统计数字卡片（教程数量、案例数量、配置教程数、覆盖工具数）
    - 数字滚动动画（进入视口时触发）
    - 每个数字配简短标签说明

- **区块**: 学习路径推荐区 (`#paths`)
  - **页面目标**: 为不同起点的用户提供清晰的学习路线
  - **功能点**:
    - 3条学习路径卡片（入门新手、进阶用户、专业开发者）
    - 每条路径含：路径名称 + 适用人群描述 + 步骤列表（3-5步）
    - 路径卡片横向排列（桌面端3列，移动端堆叠）

- **区块**: 实战案例展示区 (`#cases`)
  - **页面目标**: 展示AI工具的实际应用场景，激发学习兴趣
  - **功能点**:
    - 3-4个精选案例卡片
    - 每个案例含：场景名称 + 使用的AI工具 + 效果描述
    - 卡片式布局，简洁展示

- **区块**: 页脚信息 (`#footer`)
  - **页面目标**: 提供版权和补充信息
  - **功能点**:
    - 版权信息
    - 简洁的底部链接（如有）

## 视觉设计参考（来自 codexguide.ai 分析）

### 配色方案（适配深色主题）
| 用途 | 色值 | 说明 |
|------|------|------|
| 主强调色 | `#169c7b` / `#0f766e` | 青绿色，用于高亮、链接、按钮 |
| 背景色 | `#0d1117` / `#161b22` | 深色背景，GitHub暗色风格 |
| 卡片背景 | `#1c2128` / `#21262d` | 略浅于背景，区分卡片层级 |
| 主文字 | `#e6edf3` / `#f0f6fc` | 白色/浅灰文字 |
| 次要文字 | `#8b949e` / `#6e7681` | 中灰，描述性文字 |
| 边框 | `#30363d` | 极细边框，卡片分隔 |

### 排版风格
- 字体：系统无衬线字体栈（`-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`）
- 标题层级：大标题 2.5-3rem 粗体，区块标题 1.5-2rem，卡片标题 1.1-1.3rem
- 留白策略：区块间上下 padding 80-120px，卡片内 padding 24-32px
- 圆角：卡片 8-12px，按钮 6-8px

### 核心设计原则
- 极简克制，无多余动画（仅保留必要的hover过渡）
- 以内容阅读体验为核心
- 深色背景 + 青绿强调色 + 白色文字，程序员终端风格
- 响应式设计，移动端友好
